# vippeterhou.github.io

Personal portfolio site at www.houbitan.com, built with Hexo 6.1.0 and a custom theme called "peregrine".

## Branch Strategy

- `dev` — source files (markdown, EJS templates, Stylus CSS, JS, config). **All editing happens here.**
- `main` — compiled output only. Auto-generated by `hexo deploy`. Never edit directly.

## Development Flow

```bash
# First time setup (fresh clone or after switching branches)
rm -rf node_modules
npm install

# Start local dev server with live reload
npm run server
# → http://localhost:4000

# Edit source files, browser auto-refreshes on save

# Save source changes to dev branch (code only — does NOT update the live site)
git add .
git commit -m "your message"
git push origin dev

# It compiles the source and pushes the built output to main branch automatically.
# This is what actually updates the homepage.
npm run deploy
```

## Project Structure

```
_config.yml                        # Hexo site config (URL, theme, deploy target)
source/
  _posts/                          # Blog posts (markdown)
  about/index.md                   # About page content
  honors/index.md                  # Honors page (gallery built from theme config)
  research/index.md                # Research page
themes/peregrine/
  _config.yml                      # Nav links, stylesheet paths, honors image list
  layout/
    layout.ejs                     # Master HTML wrapper
    index.ejs                      # Homepage
    page.ejs                       # Router → delegates to _page/ templates
    _partial/                      # head, header, footer
    _page/                         # post, about, honors, 404
  source/
    css/style.styl                 # CSS entry point (imports all partials)
    js/scripts.js                  # Dark/light mode (all pages)
    js/home.js                     # Loading + typewriter motto (homepage only)
    js/nonhome.js                  # Loading fade (non-homepage)
```

## npm Scripts

| Command | What it does |
|---|---|
| `npm run server` | Local dev server with live reload |
| `npm run build` | Compile source into `public/` |
| `npm run deploy` | Build + push `public/` to `main` branch |
| `npm run clean` | Delete `public/` directory |
